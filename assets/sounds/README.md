# Sound Assets

This directory contains sound effect files for the app.

## Files needed:
- `tap.mp3` - Light tap sound for buttons
- `success.mp3` - Success/completion sound
- `error.mp3` - Error/failure sound
- `notification.mp3` - Notification received sound
- `sent.mp3` - Message/request sent sound

## Sources for free sound effects:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

## Requirements:
- Format: MP3
- Size: < 100KB each
- Sample rate: 44.1kHz or 48kHz
- Bit rate: 128kbps or higher
- Duration: < 1 second (except notification can be longer)

Note: For now, the audio service will work without files (silent mode).
Add actual sound files before production deployment.
