## Performance Metrics

### Load Times
- Chat list: < 100ms
- Message rendering: < 50ms per message
- Voice playback start: < 200ms
- File picker open: < 150ms
- Gallery load: < 200ms

### Memory Usage
- Message list: ~50 bytes per message
- Image thumbnails: Cached, max 5MB
- Voice recordings: Stored locally
- File references: Minimal (just metadata)

### Network
- Voice upload: Progressive (stream)
- File upload: Background with retry
- Image thumbnails: Lazy loading
- Gallery: Paginated (20 items)
